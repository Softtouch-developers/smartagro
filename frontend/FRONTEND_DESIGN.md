# SmartAgro PWA Frontend Design

## Overview

SmartAgro is a Progressive Web App (PWA) marketplace connecting Ghanaian farmers with buyers. The UI is inspired by food delivery apps like Uber Eats/Bolt Food, optimized for mobile-first usage with offline capabilities.

## User Roles & Navigation

### Role-Based Views

| Role | Bottom Nav Items | Key Features |
|------|------------------|--------------|
| **Buyer** | Home, Search, Cart, Messages, Profile | Browse products, cart, checkout, track orders |
| **Farmer** | Dashboard, Products, Orders, AI Assistant, Profile | Manage listings, fulfill orders, AI help |
| **Farmer (Buyer Mode)** | Home, Search, Cart, Messages, Profile | Same as buyer + mode switch banner |
| **Admin** | Dashboard, Users, Orders, Disputes, Settings | Platform management |

## 5. Offline Architecture

### IndexedDB Stores

```typescript
// Database: smartagro-db

interface DBStores {
  // User & Auth
  auth: {
    key: 'current',
    value: { accessToken, refreshToken, user }
  },

  // Products cache
  products: {
    key: 'id',
    indexes: ['category', 'sellerId', 'status'],
    value: Product
  },

  // Farmers cache
  farmers: {
    key: 'id',
    value: FarmerProfile
  },

  // Shopping cart (persisted)
  cart: {
    key: 'id',
    value: CartState
  },

  // User's orders
  orders: {
    key: 'id',
    indexes: ['status', 'createdAt'],
    value: Order
  },

  // Draft messages
  messageDrafts: {
    key: 'conversationId',
    value: { text, attachments }
  },

  // Pending sync queue
  pendingSync: {
    key: 'id',
    indexes: ['type', 'createdAt'],
    value: {
      id: string,
      type: 'PRODUCT_CREATE' | 'PRODUCT_UPDATE' | 'CART_ADD' | 'MESSAGE_SEND',
      payload: any,
      createdAt: Date,
      retryCount: number
    }
  }
}
```
